\section*{Open Research Section}
CMIP models used in this study are listed in Table S1 and model output is available through the Earth System Grid Federation (ESGF) at https://esgf-node.llnl.gov/search/cmip6/. 
Observational datasets and access are listed here; 
GPCP precipitation dataset: https://doi.org/10.5065/ZGJD-9B02, 
ERA5 dataset: https://doi.org/10.24381/cds.adbb2d47, 
NOAA surface temperature dataset: https://psl.noaa.gov/data/gridded/, 
CERES outgoing longwave radiation dataset: https://ceres.larc.nasa.gov/data/, 
ISCCP: ISCCP cloud states dataset: https://isccp.giss.nasa.gov/analysis/climanal5.html. 
The IFS\_9\_FESOM\_5 model data is available via the World Data Center for Climate (WDCC) at DKRZ: https://doi.org/10.26050/WDCC/nextGEMS\_cyc2. 
Code examples for reproducing key metric calculations and figures are available at \citet{Blackberg2025software}. 
Key metrics are available at \citet{Blackberg2025dataset}.
